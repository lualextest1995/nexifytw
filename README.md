# Getting Started

## development

1. Clone this repositories.

```
$ git clone https://github.com/lualextest1995/nexifytw.git
```

2. Install NPM.

```
$ cd nexifytw
$ npm install
```

3. Run the client and server.

```
$ npm run dev
```

4. Open Web Browser.

```
http://localhost:5173/
```
